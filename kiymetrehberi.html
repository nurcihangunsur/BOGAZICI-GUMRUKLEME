Value Guide

Customs Value of Goods

The customs value of goods is a fundamental concept in calculating customs duties and generally represents the value of the goods.

Why is the concept of "Customs Value" important?

Customs value is a crucial factor in calculating import and customs duties. Accurate declaration of customs value ensures fair taxation and the orderly conduct of trade.

How is the customs value determined?

The customs value is typically calculated using a formula that includes elements such as the selling price, transportation, and insurance costs.

How is the customs value declared to customs authorities?

The customs value is declared on the customs declaration by the importer or declarant. The declared value may be subject to verification by customs authorities.

Which exchange rate is used when determining the customs value of goods?

Generally, exchange rates are used when determining the customs value of goods. These rates can be daily rates from international financial markets or specific reference date rates.

How is the customs value of goods determined using different methods?

The customs value of goods can be determined using various methods, such as the transaction value method, deduction method, computed value method, etc. Each method has its own rules and advantages.

What does "Customs value according to the transaction value method" mean?

The transaction value method is a way of determining the customs value based on the selling price of the goods. The customs value is calculated based on the market where the goods are sold or will be sold.

What does the term "Actually paid or payable price" mean?

This term is considered when calculating the customs value according to the transaction value method. The actually paid or payable price refers to the real commercial value between the buyer and the seller.

What do "Identical goods" and "Similar goods" mean?

Identical goods refer to goods that are entirely the same in characteristics, while similar goods refer to goods with very similar characteristics. These concepts are important when calculating the customs value.

What does "Customs value according to the transaction value method for identical goods" mean?

This concept explains how the customs value is determined for identical goods when sold to different buyers or sellers. It outlines the methodology for calculating the customs value.

What does "Customs value according to the transaction value method for similar goods" mean?

This concept explains how the customs value is determined for similar goods, considering very close characteristics. The price of similar goods can be used as a reference in this method.

What does "Deduction method for customs value" mean?

The deduction method is a way of determining the customs value based on the transaction value method. Certain deductions made under specific conditions are taken into account.

What does "Computed value method for customs value" mean?

The computed value method is used when the customs value cannot be determined by other methods. In this method, the customs value is calculated considering cost elements, profit, and other factors.

What does "Fallback method for customs value" mean?

The fallback method is used when a customs value cannot be determined by other methods. In this method, if possible, the customs value determined by other methods in a previous similar period is taken as a reference.

How is the customs value of data or command-loaded information carriers imported for use in computers determined?

The customs value of such products is usually determined by considering cost elements, profit, and other factors. The importer can provide documents explaining the features and usage of these products.

What does "Royalty and License Fee" mean?

Royalty and license fee refer to the fee paid for the right to use a product or a brand. These payments can be taken into account when calculating customs value.

What happens if the rules regarding the determination of the customs value of goods are not followed?

If incorrect or incomplete information is provided in the customs value declaration, customs authorities may conduct an examination. Providing false information may result in penalties or tax violations.

What is an Exceptional Value Declaration? How is it done?

An exceptional value declaration refers to a situation where the customs value declaration is not made under certain conditions or where the customs value is determined by a specific regulation. These situations generally cover exceptional and special circumstances.
